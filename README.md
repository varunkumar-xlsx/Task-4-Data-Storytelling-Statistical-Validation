# 📊 Task 4 — Data Storytelling & Statistical Validation

This task focuses on performing advanced data analysis and validating insights using statistical hypothesis testing. The objective is to move beyond descriptive analytics and evaluate data-driven assumptions using Python.

The analysis was performed on the sales dataset used in previous tasks. Along with business insights, statistical testing was applied to verify whether certain patterns observed in the data are statistically significant.

🧰 Tools Used
- Python (Pandas, SciPy)
- Google Colab / Jupyter Notebook
- PowerPoint
- Data Visualization concepts

# Key components of Task 4
1. Data Analysis
The dataset was analyzed to identify patterns related to sales performance, customer segments, and regional distribution.
2. Hypothesis Testing
A statistical hypothesis test was conducted to validate assumptions derived from the data.
Null Hypothesis (H₀): There is no significant difference between the compared groups.
Alternative Hypothesis (H₁): There is a significant difference between the compared groups.
Python libraries were used to compute the statistical test and interpret the results.
3. Result Interpretation
Based on the p-value obtained from the statistical test, the hypothesis was evaluated to determine whether the null hypothesis should be accepted or rejected.
